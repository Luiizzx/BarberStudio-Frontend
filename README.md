# FrontEnd do projeto BarberStudio

Esse FrontEnd foi feito utilizando React JS, com Node na versão 25.6.1.

Para executar a aplicação, primeiro instale o Node através do NVM.

O NVM pode ser baixado no link abaixo (desça a página e procure "nvm-setup.exe"):

https://github.com/coreybutler/nvm-windows/releases

Faça a instalação normal e, no terminal, digite:

>nvm install node

Acesse a pasta onde clonou o repositório pelo terminal e digite:

>npm install

E depois:

>npm run dev

Você pode visualizar a aplicação clicando no link que aparece no CMD, onde a aplicação está rodando.
